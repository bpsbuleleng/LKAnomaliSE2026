Kelola Rule

Satu rule = satu anomali: message yang tampil + kondisi when pendeteksinya. Nonaktifkan = rule berhenti dijalankan saat submit, baris tidak dihapus.

Tambah Rule

Boleh diubah kapan saja (huruf/angka/underscore/strip), mis. K_status_cerai — kosongkan saat menambah rule baru untuk id otomatis. Mengubah ID tidak memengaruhi anomali yang sudah tersimpan di record lama.

Kombinator: all/any (bersarang), roster_any/roster_all/roster_count + condition. Operator: ==, !=, >, >=, <, <=, in, not_in, empty, not_empty, regex. Bandingkan antar-field pakai "field2".
